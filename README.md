<h1 align="center">🙋‍♀️ EMPLEADAS-API 🙋‍♀️</h1>

Creacion de una pequeña API Rest que permite realizar la alta, baja y modificacion de empleadas y sus respectivas categorias.

<h2>🔨 Creado usando:</h2>

- JAVA 11.
- SpringBoot.
- Maven.
- Hibernate.
